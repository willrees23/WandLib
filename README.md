<h2>Included</h2>
<ul>
  <li><a href="https://github.com/dejvokep/boosted-yaml">BoostedYaml</a> (custom configuration & data files)</li>
  <li><a href="https://github.com/JorelAli/CommandAPI">CommandAPI</a> (custom commands without touching plugin.yml)</li>
</ul>
<h2>Setup</h2>
Add the following to your plugin's pom.xml.
<br></br>

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
```xml
<dependency>
    <groupId>com.github.willrees23</groupId>
    <artifactId>WandLib</artifactId>
    <version>[insert version]</version>
</dependency>
```
